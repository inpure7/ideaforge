IdeaForge · 아이디어 씨앗을 검증된 사업계획으로 키워주는 AI 창업 코파일럿
요약 특허 상세 시장/전략 상세 기획서 QA MVP 목업
기획서(무엇을 했는지)
  • 제품명: IdeaForge
  • 타겟: 예비 창업자, 1인 창업자(Solo Founder), 사이드 프로젝트를 진행하는 직장인, 대학생 창업 준비자, 기업 내 이노베이션팀
  • 문제: 대부분의 예비 창업자는 아이디어는 있지만 시장성 검증, 린캔버스 작성, MVP 정의까지 이어지는 구조화된 프로세스가 없어 아이디어 단계에서 포기하거나 방향 없이 시간을 낭비한다. 기존 액셀러레이터(YC 등)는 합격률 1~2% 미만으로 접근성이 극히 낮고, 범용 AI(ChatGPT 등)는 창업 전용 프레임워크가 없어 단편적인 답변만 제공한다.
  • 솔루션: 창업 아이디어 전용 AI 검증 프레임워크를 내장한 SaaS 플랫폼. 사용자가 아이디어를 입력하면 ① AI 기반 시장성 분석(TAM/SAM/SOM 자동 추정) ② 린캔버스 자동 생성 ③ 경쟁사 맵핑 ④ MVP 기능 스펙 로드맵 ⑤ 커뮤니티 피어 리뷰까지 원스톱으로 제공. 아이디어 버전 관리로 진화 과정을 추적하고, 한국어 특화 국내 시장 데이터를 연동해 로컬 맥락을 반영한다.
  • 비즈니스 모델: Freemium SaaS 구독 모델을 기반으로, 무료 플랜으로 사용자를 확보한 뒤 Pro 구독 및 B2B 엔터프라이즈 계약으로 수익을 창출한다. 추가적으로 단건 심층 분석 리포트 판매와 전문가 멘토링 연결 수수료로 수익을 다각화한다.
MVP 기능
  • 아이디어 입력 → AI 자동 린캔버스 생성 (문제/솔루션/USP/채널/수익모델 9개 블록 자동 완성)
  • TAM/SAM/SOM 시장 규모 자동 추정 리포트 + 경쟁사 3개 자동 식별 및 차별점 분석
  • MVP 핵심 기능 우선순위 로드맵 자동 생성 (Must-have / Nice-to-have 분류)
  • 아이디어 버전 히스토리 관리 (수정 이력 추적 및 버전 간 비교)
  • 커뮤니티 피어 리뷰 (익명 또는 실명으로 동료 창업자에게 피드백 요청)
기술 스택
frontend: React + Next.js (SSR, SEO 최적화) + TailwindCSSbackend: FastAPI (Python) + LangChain (AI 오케스트레이션) + Celery (비동기 리포트 생성)db: PostgreSQL (사용자·아이디어 데이터) + Redis (캐싱·세션) + Pinecone (아이디어 벡터 임베딩·유사도 검색)
수익 구조
  • Free 플랜: $0/월 — 월 3개 아이디어, 기본 린캔버스 생성 (사용자 유입 및 바이럴 채널)
  • Pro 플랜: $19/월 — 무제한 아이디어, 전체 AI 분석 리포트, TAM/SAM/SOM 자동화, 피어 리뷰 무제한
  • Enterprise 플랜: $299/월~ (팀 10인 이상) — 사내벤처·대학 창업지원단 전용 대시보드, API 연동, 전담 CS
  • Pay-per-Report: 건당 $9 — 심층 시장 분석 리포트 (경쟁사 분석, 특허 리스크, 규제 환경 포함)
  • 멘토·전문가 연결 중개 수수료: 세션당 15% 수수료
  • B2G 계약: 중소벤처기업부·창진원 연계 정부 창업 프로그램 공급
차별점
  • 창업 전용 AI 프레임워크: 범용 AI와 달리 린캔버스·TAM/SAM/SOM·MVP 로드맵이 단일 워크플로우로 연결되어 '아이디어 입력 → 사업계획 초안' 까지 10분 내 완성
  • 아이디어 버전 관리 & 진화 추적: Git처럼 아이디어 수정 이력을 관리하고 버전 간 차이를 시각화 — 기존 어떤 경쟁사도 제공하지 않는 기능
  • AI + 커뮤니티 하이브리드 검증: AI 단독 검증의 신뢰성 한계를 동료 창업자 피어 리뷰로 보완, 정성적·정량적 피드백 동시 제공
  • 한국어 특화 + 국내 시장 데이터 연동: 국내 산업 통계, 공공 데이터 API 연동으로 글로벌 플랫폼이 제공하지 못하는 한국 로컬 맥락 반영
  • YC 수준 코칭의 민주화: 합격률 1~2% 액셀러레이터 대신 누구나 즉시 구조화된 아이디어 검증 프로세스에 접근 가능
리스크
  • 범용 AI 대체 위험: ChatGPT·Claude의 기능 고도화로 단순 아이디어 분석이 무료화될 경우 차별화 약화 → 대응: 창업 전용 데이터 축적과 커뮤니티 네트워크 효과로 진입장벽 구축
  • AI 검증 신뢰성 문제: 잘못된 시장 분석 결과로 사용자가 잘못된 의사결정을 내릴 경우 신뢰 손상 → 대응: AI 결과에 '불확실성 지표' 명시, 피어 리뷰로 교차 검증 필수화
  • 아이디어 데이터 보안 우려: 창업 아이디어 유출에 대한 사용자 민감도가 높아 가입 거부 요인 → 대응: 엔드투엔드 암호화, 비공개 설정 기본값, SOC2 인증 획득 로드맵 공개
  • 유료 전환율 저조 위험: 무료 대안(Notion + AI) 대비 유료 결제 설득력 부족 → 대응: 무료 플랜 기능을 의도적으로 제한하고 Pro 전환 유도 인앱 트리거 설계
  • 스타트업 투자 시장 위축: 투자 혹한기 지속 시 예비 창업자 수 감소 → 대응: B2B(기업 이노베이션팀) 및 B2G(정부 프로그램) 채널로 수요 분산
마일스톤
  • Week 1 — 핵심 팀 구성 완료 (PM, AI 엔지니어, 풀스택 개발자, 디자이너) 및 기술 스택 확정. 경쟁사 5개 상세 분석 및 차별화 포인트 문서화
  • Week 2 — 아이디어 입력 → 린캔버스 자동 생성 AI 파이프라인 프로토타입 완성 (LangChain + GPT-4o 기반). 핵심 프롬프트 엔지니어링 v1 완료
  • Week 3 — TAM/SAM/SOM 자동 추정 모듈 개발 및 경쟁사 자동 식별 기능 구현. 프론트엔드 와이어프레임 완성 및 사용자 플로우 확정
  • Week 4 — 아이디어 버전 관리 시스템 개발 완료. 내부 알파 테스트 실시 (팀 + 지인 10명). 주요 UX 이슈 도출 및 수정
  • Week 6 — MVP 기능 5개 전체 통합 완료. 클로즈드 베타 사용자 50명 온보딩. NPS 및 핵심 지표(아이디어 생성 완료율, 세션 시간) 측정 시작
  • Week 8 — 베타 피드백 반영한 v1.1 배포. Product Hunt 론칭 준비 완료 (소개 영상, 랜딩페이지, 헌터 확보). 대학 창업지원센터 2곳 파일럿 미팅
  • Week 10 — Product Hunt 공개 론칭. 목표: 가입자 500명, Product of the Day Top 5 진입. 한국 스타트업 커뮤니티(디스콰이엇, 스타트업 얼라이언스) 동시 홍보
  • Week 12 — Pro 플랜 유료 구독자 50명 달성 (MRR $950 목표). 대학 창업지원센터 1곳 B2B 파일럿 계약 체결. 월간 활성 사용자(MAU) 300명 이상 확보
